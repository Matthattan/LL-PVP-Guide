Welcome to the unofficial beginner's guide to PVP!

The Aim of this website is to aid players stepping into Loomian Legacy's Competitive Scene. This website addresses how to Teambuild, build pvp Loomians, understanding cores and the different facilities (both ingame and online) that can help in your journey.

This is an unofficial guide not endorsed by the Loomian Legacy developers but rather created by the community. We strive for information on this website to be correct and up to date; if there is any incomplete or incorrect information feel free to let us know.
